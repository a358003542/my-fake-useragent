# my-fake-useragent
generate your random fake useragent.

## install
```
python setup.py install
```

or
```text
pip install my_fake_useragent
```
    
## usage

```python
from my_fake_useragent import UserAgent

ua = UserAgent(family='chrome')

res = ua.random()

```

### family参数可用：
可以设置列表还匹配多种情况

- chrome
- firefox
- edge
- ie
- opera
- safari

### os_family参数可用
可以设置列表还匹配多种情况

- android
- windows
- linux
- mac
- ios
- chrome os

### phone参数可用

- None 不过滤
- True 要求是移动端
- False 要求不是移动端
